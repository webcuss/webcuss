# webcuss | extension

### Run dev
```shell
$ export REACT_APP_BACKEND_URL=http://localhost:8080
$ npm install
$ npm start
```

### Build
```shell
$ export REACT_APP_BACKEND_URL=http://localhost:8080
$ npm run build
```

### Import extension
Goto Google Chrome > Extension > Enable `Developer Mode` > Load unpacked > select `webcuss/extension`
